# miao-plugin-ProfileImg
这个图库是 Yunzai 框架下的 miao-plugin 面板图资源，涵盖了**原神&星铁**的面板图资源

## 安装方式

> 在Yunzai根目录执行命令安装
Github
``` bash 
git clone https://github.com/AxiuCN/miao-plugin-ProfileImg ./plugins/miao-plugin/resources/profile/normal-character
```
