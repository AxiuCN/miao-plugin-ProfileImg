# miao-plugin-ProfileImg
这个图库是 Yunzai 框架下的 miao-plugin 面板图资源，涵盖了**原神&星铁**的面板图资源

## 安装方式

> 在Yunzai根目录执行命令安装
```bash 
//使用Github
cd plugins/miao-plugin/resources/profile/normal-character
git init
git remote add origin https://github.com/AxiuCN/miao-plugin-ProfileImg.git
git fetch origin main
git reset --hard origin/main
```

## 面板图图库管理器
用于管理本图库的单文件插件，支持查看状态、自动/手动更新等功能

### 安装
在 Yunzai 根目录下执行以下命令：

```bash
curl -o plugins/example/面板图图库管理器.js "https://raw.githubusercontent.com/AxiuCN/Yunzai_JS_Plugins/main/面板图图库管理器.js"
```
